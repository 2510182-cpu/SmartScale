About the Project

Ye project aik Smart IoT Kitchen Weighing Machine hai jo NodeMCU ESP8266 aur HX711 Load Cell sensor ka istemal karti hai. Iska maqsad kitchen mein cheezon ka wazan (weight) bilkul accurate tareeke se mapna hai.

Is project ki khasiyat ye hai ke ye Real-time Data ko Google Firebase Cloud par bhejta hai, jis se aap dunya mein kahin bhi baith kar apni website par live weight aur button ka status dekh sakte hain. Ye modern kitchen automation ki taraf ek chota sa qadam hai.
