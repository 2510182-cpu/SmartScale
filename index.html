Project Overview

This Smart IoT Kitchen Scale is a cloud-integrated measuring system designed to modernize the cooking experience. Built using the NodeMCU ESP8266 and the HX711 Load Cell sensor, it provides high-precision weight measurements.

The system utilizes Google Firebase for real-time data synchronization, allowing users to monitor weight and hardware status remotely via this web dashboard. This project demonstrates the practical application of Internet of Things (IoT) in everyday home automation and smart kitchen environments.
